HTML Structure -top

This theme divided into many parts, each parts content html comment where started and where finished. Each theme contain in different files. The html file name indicate which theme you're chossing to use

CSS Structure -top

Each theme contains different style files. So if you want to change or apply any style make sure to check the style attached to html file, then there also bootsrap4 style added on the theme. The style is choronological order and each part of the style contains comment that which area the style belongs to.

JavaScript Structure -top

In main.js file contain the scrolling effects of webpage. The preloader.js contain the preloader of webpage. Quill.js is a reach text editor for Blog Dashboard.

Particle -top

You can customize the particle theme in particle.json file which is available in js folder. Check out the official particle website to customize it.
